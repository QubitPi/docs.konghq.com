Open API Specifications

Welcome to Kong API Specs

This section of our docs site is Powered by Konnect, using customizations available for the DevPortal.

Definitions

Gateway Dev Portal
Dev Portal provides a single source of truth for all developers to locate, access, and consume services for Kong Gateway EE. This spec is a Beta version.
Latest Version: v1 Specification
Gateway Admin - EE (Beta)
Kong Gateway (EE) comes with an internal RESTful API for administration purposes. This spec is a Beta version.
Latest Version: 3.5.0.0 Specification
Gateway Admin - OSS (Beta)
Kong Gateway (OSS) comes with an internal RESTful API for administration purposes. This spec is a Beta version.
Latest Version: 3.5.0 Specification
Gateway Status
Gateway Status is a read-only endpoint allowing monitoring tools to retrieve metrics, healthiness, and other non-sensitive information of the node.
Latest Version: v1 Specification
Konnect API Products
API Products can be used to manage API versions, documentation, and analytics.
Latest Version: v2 Specification
Konnect API Requests (Beta)
API Requests provides access to detailed records for requests made to your API, in near real-time. This API is released as beta.
Latest Version: v2 Specification
Konnect Audit Logging
Audit logging enables administrators to better spot security risks and maintain compliance of their core infrastructure.
Latest Version: v2 Specification
Konnect Control Planes Config (Beta)
Manage Kong Gateway services, routes, certificates, consumer and global configuration within Konnect. This spec is a Beta version.
Latest Version: v2 Specification
Konnect Control Planes
Manage the control plane entity in Konnect.
Latest Version: v2 Specification
Konnect Identity Management
Identity Management is responsible for Konnect users, teams, organization, roles, permissions, and IdP settings.
Latest Version: v3 Specification
Konnect Mesh Manager
The Mesh Manager API can be used to create and manage your global mesh control planes.
Latest Version: v0 Specification
Konnect Customizable Portal
Customize your self-hosted developer portal experience.
Latest Version: v2 Specification
Konnect Portal Management
Integrate portal management operations with automation systems (such as CI/CD pipelines) utilizing this AIP-compliant API.
Latest Version: v2 Specification
Konnect Portal RBAC
Portal RBAC can be used to create teams, collect developers into teams, and assign roles to give those teams functionality in the developer portal.
Latest Version: v2 Specification
